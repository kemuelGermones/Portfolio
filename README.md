### Requirements

- React.js

### Usage

Clone it

```
$ git clone git@github.com:kemuelGermones/Porfolio.git
```

Go into the directory and install all dependencies

```
npm install
```

Run the command in the directory:

```
npm run dev
```

Open the url specified in the terminal and enjoy!
